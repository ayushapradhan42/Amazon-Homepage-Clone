# Amazon Clone (HTML & CSS)

This is a basic Amazon homepage clone that I built using only HTML and CSS.  
I made this project to practice layout design, styling, and improving my frontend skills.

## Features
- Header and navigation bar similar to Amazon
- Search bar UI
- Product section layout
- Footer with multiple link sections
- Responsive layout (basic)

## Technologies
- HTML
- CSS

## How to Run
Just download the files and open `index.html` in your browser.
